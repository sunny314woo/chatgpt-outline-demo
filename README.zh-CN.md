# ChatGPT Outline (Demo)

本目录包含 ChatGPT Outline 的精简 Demo 版本。

## 概览

这是一个仅在本地运行的浏览器扩展，专注做一件事：

- 从 ChatGPT 对话中生成清晰的目录
- 点击目录项后跳转到对应消息
- 在对话变化时自动刷新目录
- 提供固定宽度、可左右切换的侧边栏

## Demo 版包含

- 从当前 ChatGPT 页面提取目录
- 点击目录跳转
- 跳转后的临时高亮
- 自动刷新目录
- 固定宽度侧边栏
- 侧边栏左右切换

## Demo 版不包含

- 激活、授权、试用、支付或远程权限逻辑
- 支付或远程权限校验
- 重点标记或重点视图
- 导出或 Markdown 导出
- Inbox 或其他外部集成
- 商业版相关服务

## 定位说明

这是一个简单、可查看源码、专注目录导航的 source-available 版本。

随着使用，我自己做了一个更完整的版本，增加了一些功能，例如：

- 重点标记
- 对话导出
- 更流畅的长对话导航

这些高级和商业化功能不包含在当前 Demo 版中。

一开始，这只是我给自己做的一个小工具。

## 隐私与数据处理

- 聊天内容仅在浏览器本地处理
- 当前 Demo 版不会把对话内容上传到远程服务器
- 当前 Demo 版不包含远程权限校验、云同步或分析统计
- 浏览器存储仅用于保存侧边栏位置等轻量本地偏好

## 本地开发

可将本目录作为未打包扩展加载到 Chrome / Chromium 中。

所有功能均在浏览器本地运行。
